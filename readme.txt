This project is implementation trading place for cryptocurrency.There are a few roles in this project: admin, user, and securities commission representative.
The admin provides approvement deposit of funds. The securities commission representative can blocks trading pair. The user, in turn, can to do deposit and
to do withdraw. There are three trading pair in this market which users can trades. The following functions are available to the user: user can set a few types
 of order(limit,market), also user can rejects this orders.One of the users installs the application(limit order) and the other user executes this application
(market order).
